RULES

1. Anda harus login sebelum bermain.
2. Setelah login, Anda diminta untuk memilih salah satu dari 3 pokemon dari Professor Oak! Pokemon yang tersedia yaitu: Sprigatito, Fuecoco, Quaxly
3. Peraturan saat Battle :
WIN Jika :
  • Pokemon Api dapat mengalahkan Pokemon Daun
  • Pokemon Daun dapat mengalahkan Pokemon Air
  • Pokemon Air dapat mengalahkan Pokemon Api
DRAW Jika :
  • Pokemon Api dapat mengalahkan Pokemon Api
  • Pokemon Daun dapat mengalahkan Pokemon Daun
  • Pokemon Air dapat mengalahkan Pokemon Air
4. Battle dilakukan selama 3 pertandingan sampai dapat 3 kali WIN, jika terdapat DRAW maka akan diulang pertandingannya
