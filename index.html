RULES

  1. Anda harus login sebelum bermain.
  2. Setelah login, Anda diminta untuk memilih salah satu dari 3 pokemon dari Professor Oak! Pokemon yang tersedia yaitu: Sprigatito, Fuecoco, Quaxly
  3. Peraturan saat Battle :
    • WIN Jika :
      1. Pokemon Api dapat mengalahkan Pokemon Daun
      2. Pokemon Daun dapat mengalahkan Pokemon Air
      3. Pokemon Air dapat mengalahkan Pokemon Api
      DRAW Jika :
      1. Pokemon Api dapat mengalahkan Pokemon Api
      2. Pokemon Daun dapat mengalahkan Pokemon Daun
      3. Pokemon Air dapat mengalahkan Pokemon Air
  4. Battle dilakukan selama 3 pertandingan sampai dapat 3 kali WIN, jika terdapat DRAW maka akan diulang pertandingannya
  5. Jika dalam 3 pertandingan anda menang maka akan tampil "Selamat Trainer Anda memenangkan Gym Leader Cynthia!" üèÜ
